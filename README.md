# learn_vue

VueJS 2 & 3 Learning projects.

### Recommended VSCode Extensions

- **Vue VSCode Snippets**: Useful Snippets for Vue 2 & 3.
- **Vetur**: Vue tool for Vue coloring and formatting.
- **Prettier**: VSCode Code formatter (supports other JS varieties, including HTML, YAML, Markdown and CSS)
- **Path Intellisense**: System Filename autocompletion for module imports (ES6, CommonJS, etc).
